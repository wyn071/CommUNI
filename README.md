**CommUNI** is a Tinder-like club matching mobile application developed as a prototype to simplify how university students (and faculty) connect, discover, and engage within their campus communities. 

# What can CommUNI do?
The app integrates academic information input and a personality test upon successful registration, and the results are used as the basis for the types of clubs or organizations suggested on the discovery page.
